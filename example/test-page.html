Implementing Surfly Javascript

1. Enter your Widget Key &
Load Surfly JS from Selected Environment
Integrations > Widget Key

2. Select Additional Options
View the Page Source to see the JS used

add floating "share screen" button

3. Enter a valid URL &
Click Start Session to begin
The Surfly session URL will be loaded into the iframe below.
Use the footer buttons to control the session with JS.

Communicating with the Surfly iFrame

To test communication between the Surfly iframe and the parent page. Load the URL below and click the button

https://demo.surfly.com/resources/send-message-to-parent.html

A line will post in the log to confirm the click. This function is useful when you need to establish a communication channel between your JS code on the original page and the Surfly session iframe
