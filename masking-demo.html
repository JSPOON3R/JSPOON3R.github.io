Surfly Masking

What's possible?

This page aims to showcase the different masking possibilities offered by Surfly. Each entry includes an overview of how it works and an outline what is required to get started

Field Masking

Ensure text entered into a specified field is not visisble to other participants

Required changes to website code? Yes

What can it mask? Text input fields
