Different ways of keeping content hidden from other participants

Field Masking

Ensure text entered into a specified field is not visisble to other participants

What can it mask? Text input fields


Element Masking

Easily mask any page elements with Surfly's default masking colors

What can it mask?: All page elements (e.g. Text, Buttons, Images, iFrames, etc)


Custom Style Element Masking

Create your own styles for masked page content

What can it mask?: All page elements (e.g. Text, Buttons, Images, iFrames, etc)


Control Dependent Appearance
